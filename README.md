**BookNest — Web Application for Books and Reviews**

BookNest is a web application for managing books and user reviews, supporting CRUD operations and admin moderation to ensure organized and high-quality content. Users can explore, review, and filter books by genre or author, while administrators control what content is published.
________________________________________
**Overview**
BookNest allows users to:
•	Browse and search books by title, genre, or author
•	View book descriptions and cover images
•	Submit reviews
Administrators can:
•	Approve or reject user-submitted books and reviews
•	Edit or remove content to maintain quality
________________________________________
**Features**
•	CRUD operations: Create, read, update, and delete reviews and books (admin-controlled)
•	Genre & author filtering: Easily find books by type or author
•	Book description & cover pages: Detailed info and visuals for each book
•	Content moderation: Admin approval ensures verified content
________________________________________
**Tech Stack**
•	Frontend: HTML, CSS, JavaScript
•	Backend & Database: SQL with role-based access control
•	Tools: Git for version control, tested in modern browsers
